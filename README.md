# KPI Dashboard - SLA Target Setting

   ## Project Overview
   Interactive Streamlit dashboard analyzing customer support ticket resolution performance by agent experience level. Provides data-driven insights for setting realistic SLA targets.

   ## Key Findings
   - Junior agents: 33.90 hours average resolution
   - Mid-level agents: 36.62 hours average resolution
   - Senior agents: 36.59 hours average resolution
   - Total tickets analyzed: 1,289

   ## Technologies Used
   - Python
   - Streamlit
   - Pandas
   - Plotly
